# Pulse

A comprehensive event management system for university fests.

## Features

-   **Dashboard**: Overview of event stats.
-   **Map**: Location-based event discovery.
-   **Assistant**: AI-powered helpdesk (powered by Gemini).
-   **Alerts**: Real-time notifications for event updates and emergencies.
-   **Profile**: User information and settings.
-   **Volunteer Management**: Assign and track volunteer tasks.
-   **Organizer Tools**: Manage events and participants.
-   **Accessibility**: Enhanced support for users with disabilities.
-   **PWA**: Installable web application with offline support.
